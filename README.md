# üêæ Pet Adoption Landing Page

This is a responsive pet adoption landing page built using **HTML and CSS**.

I created this project to practice layout design, background images, gradients, and responsive design using media queries. The focus was on improving my frontend fundamentals and understanding how to structure multi-section websites properly.



##  Features

 Responsive full-screen Hero section
 Separate sections for different pets (Tom & Jerry)
 Background images with gradient overlays
 Clean navigation bar
 Call-To-Action (CTA) section with contact details
 Media queries for mobile responsiveness
 Smooth scrolling


##  Tech Used

 HTML5
 CSS3
 Flexbox
 Media Queries
